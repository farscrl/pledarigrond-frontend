Installation der Rechtschreibkontrolle in Microsoft 365

Microsoft erlaubt leider keine Installation von rätoromanischen Sprachpaketen, die es ermöglichen würden, die romanischen Idiome wie andere Sprachen zu überprüfen. Wir haben aber ein Add-in entwickelt, das die rätoromanische Rechtschreibprüfung in Microsoft Word zur Verfügung stellt.

Schritte, um das Add-in zu installieren:

  1. Word öffnen
  2. Gehe zu «Einfügen» -> «Add-ins (installieren)» -> «Store»
  3. «rumantsch» ins Feld tippen und suchen
  4. Neben «Controlla ortografica rumantscha» auf die Schaltfläche «Hinzufügen» klicken
  5. Zum Tab «Start» zurückkehren (es kann einen Augenblick dauern bis die Installation abgeschlossen ist)
  6. Auf «Ortografia rumantscha» klicken
  7. In der Seitenleiste, die sich geöffnet hat, unten rechts auf «Parameters» klicken
  8. Ein Idiom auswählen
  9. Zum Reiter «Correctura» zurückkehren und die Anweisungen lesen

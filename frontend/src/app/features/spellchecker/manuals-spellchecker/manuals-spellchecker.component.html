Installation der Rechtschreibprüfung auf macOS

Die romanische Rechtschreibprüfung basiert auf der Software «Hunspell», die auch von macOS genutzt wird. Wenn das romanische Wörterbuch direkt in macOS installiert wird, steht die Rechtschreibkorrektur in einigen Apple-Programmen, unter anderem Mail und TextEdit zur Verfügung. Allerdings müssen die Dateien manuell installiert werden.

Mit folgenden Schritten installieren Sie die Rechtschreibprüfung für das Surmiran auf macOS:

  1. Laden Sie folgende Datei herunter:

  2. Entpacken Sie die Zip-Datei

    Durch Doppelklick auf die Zip-Datei wird der Ordner entpackt, der vier verschiedene Dateien enthält.

    - eine Datei mit dem Wörterbuch
    - eine Datei mit Regeln zur Generierung von weiteren Wörtern
    - eine Datei mit der Lizenz
    - eine Datei mit Angaben zur Version

  3. Öffnen Sie den Ordner ~/Library/Spelling im Finder

    Öffnen Sie zu diesem Zweck den Finder oder einen beliebigen Ordner, dann das Menü «Gehe zu» und klicken Sie dann auf «Gehe zum Ordner».

    Geben Sie dann «~/Library/Spelling» (ohne Anführungszeichen!) ein und klicken Sie auf «Öffnen».

  4. Kopieren Sie die vier Dateien in den geöffneten Ordner

  5. Schliessen Sie alle Fenster und starten Sie den Computer neu

    Sobald man in der entsprechenden Software «Rumantsch (Library)» ausgewählt hat, steht nun die romanische Rechtschreibprüfung zur Verfügung.
